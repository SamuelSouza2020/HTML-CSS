Dentro da tag meta os atributos permitidos são:

TAG HTTP-Equiv

  • Cache-Control - Determina o comportamento da página
  • Pragma no-cache - Não armazena a página em cache
  • Content-Language - Declaro o idioma utilizado
  • Expires - Data de validade
  • Refresh - Tempo para atualizar
  • Imagetoolbar - Elimina barra de opões
  • Content-Script-Type - Tipo de linguagem script
